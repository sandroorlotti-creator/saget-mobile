⚙️ Configurazione

Inserisci i dati del tuo Firebase Realtime Database per sincronizzare l'agenda con SaGest sul PC.

Come configurare Firebase (gratuito):
1. Vai su console.firebase.google.com
2. Crea un nuovo progetto (es. "SaGest")
3. Vai su Realtime Database → Crea database
4. In Regole imposta read/write = true
5. Copia l'URL del database qui sotto
